# References

This directory is reserved for bibliographic notes, literature summaries, and source-specific research materials that support official experiments and methods documentation.

Reference material added here should distinguish clearly between:

- primary scientific literature,
- methodological references for numerical or quantum-computational techniques,
- historical or archival materials incorporated later for context.

The repository-level Phase 6 citation policy is defined in [citation-and-bibliography-policy.md](citation-and-bibliography-policy.md).
